# universe/

Star Citizen celestial bodies + POI catalog, derived from publicly-released CIG game files.

The bundled `dataset-v<patch>.json` (~12 MB) is published as a [GitHub Release](https://github.com/zimm1/sc-agent-public/releases?q=dataset) for the sc-agent app to consume. This directory currently holds only this README; per-system human-friendly browsing splits will be added when the maintainer finds the time.

For schema + lookup patterns, see [`/docs/data-format.md`](../docs/data-format.md).

Latest release: [`dataset-v4.7.2`](https://github.com/zimm1/sc-agent-public/releases/tag/dataset-v4.7.2).
